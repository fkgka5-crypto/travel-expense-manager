# travel-expense-manager

업무 자동화 및 연구과제 관리 프로세스 개선을 위한 실용적인 출장비 정산 및 증빙 관리 시스템(Travel Expense Calculation and Reimbursement Management System)입니다. 

본 프로젝트는 출장 프로세스에서 발생하는 반복적인 행정 업무를 자동화하고, 증빙 관리의 효율성과 보안성을 높이기 위해 개발되었습니다.

---

## 🚀 주요 기능 (Key Features)

- **출장비 자동 정산:** 일비, 숙박비, 교통비(KTX, 항공 등)를 기준에 맞게 계산하여 총 출장비를 자동으로 산출합니다.
- **증빙 자료 검증 및 보안:** 영수증 파일(JPG, PNG, PDF 등)의 확장자와 무결성을 검증하여 잘못된 파일 업로드 및 보안 위협을 차단합니다.
- **프로세스 자동화:** 복잡한 연구과제 관리 및 행정 증빙 절차를 간소화하여 업무 효율성을 증대시킵니다.

---
